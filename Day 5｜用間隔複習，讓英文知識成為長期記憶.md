# Day 5｜用間隔複習，讓英文知識成為長期記憶

在 Day 4 學習了主動回憶法後，我們已經知道如何透過測驗來檢驗學習效果，確保真正掌握單字而不是只有「看起來熟悉」的假象。但光是知道「如何有效學習」還不夠，更關鍵的問題是：「什麼時候複習最有效？」

今天我們要深入探討**記憶曲線**的科學原理，並學習如何運用間隔複習讓你的英文知識從短期記憶轉化為長期記憶。

**記憶曲線**由德國心理學家 Hermann Ebbinghaus 在 1885 年提出，描述人類學習後記憶的自然衰退規律。它揭示了一個關鍵事實：記憶在最初幾天衰退最快，之後逐漸趨緩。

<img src="./images/AI-day5-1.png" alt="Hermann Ebbinghaus記憶曲線圖" style="width: 60%; max-width: 700px;">

*▲ Hermann Ebbinghaus記憶曲線圖*

**如圖所示**，紅色的曲線代表的就是記憶自然衰退的過程，它顯示了記憶量會隨著時間快速下降。而綠色的曲線則代表「間隔複習」的效果，它說明了每一次的複習都能將記憶曲線重置，並且隨著複習次數的增加，曲線會變得越來越平緩，代表記憶也變得越來越穩固。

這張圖清晰地說明了，**當主動回憶遇上記憶曲線，就形成了科學化的學習系統**：在記憶即將衰退的關鍵時間點，透過主動回憶來重新強化記憶。

## 利用Anki打造記憶曲線實作

理論很美好，但如何實際執行間隔複習呢？手動計算複習時間太複雜，這時候我們需要一個自動化的工具——**Anki**。

### 什麼是 Anki？

Anki 是一個基於間隔重複算法的記憶卡軟體，它會根據你每次回答的表現自動調整下次復習的時間間隔。當你答對時，間隔會拉長；答錯時，間隔會縮短，讓你在「即將忘記」的時候進行復習。

### Anki 的核心優勢

1. **自動計算最佳複習時間**：不需要手動記錄，系統會根據記憶曲線自動安排
2. **個人化學習節奏**：每個人的記憶能力不同，Anki 會根據你的表現調整
3. **跨平台同步**：手機、電腦、網頁版都能使用，隨時隨地複習
4. **豐富的卡片格式**：支援文字、圖片、音頻，完美配合記憶錨點理論

### Anki 實作步驟

#### 步驟一：下載與安裝 Anki

1. 到官網 [ankiweb.net](https://ankiweb.net) 下載軟體
2. 註冊 AnkiWeb 帳號（用於同步）
3. 在手機上下載 AnkiApp（注意：選擇官方版本）

#### 步驟二：建立你的第一副學習卡組

假設我們要把 Day 3 學習的健身房單字製作成 Anki 卡片：

**單字清單**：gym, treadmill, dumbbell, stretch, sore

**建立步驟**：
1. 點選「建立卡組」
2. 命名為「健身房英文」
3. 選擇「新增」來建立卡片

#### 步驟三：設計有效的 Anki 卡片

結合前面學習的理論，我們來設計一張完整的 Anki 卡片：

**正面（問題面）**：
```
[情境引導] 我在健身房看到這個器材：
[圖片錨點] （插入啞鈴圖片）
這個英文單字是？
```

**背面（答案面）**：
```
**Dumbbell** /ˈdʌmˌbɛl/

📍 記憶錨點：
• 圖像：沉重的金屬啞鈴在健身房架上
• 情境：第一次舉啞鈴時手臂顫抖的感覺
• 故事：Dumb（啞的）+ bell（鈴），像是不會響的鈴鐺

💪 例句：
"I started with 5kg dumbbells and worked my way up."
（我從 5 公斤的啞鈴開始，逐步增加重量。）

🔗 語意網路：
• 相關：barbell（槓鈴）, weights（重量訓練）
• 動作：lift（舉起）, curl（彎舉）
• 場所：gym（健身房）, fitness center（健身中心）
```

#### 步驟四：設定最佳複習參數

在 Anki 中調整以下設定：

**學習階段**：
- 新卡片：1分鐘 → 10分鐘 → 4小時 → 1天
- 復習間隔：初始 1天，成功後逐漸延長到數月

**每日限制**：
- 新卡片：5-10張（避免負擔過重）
- 復習卡片：無限制（必須完成當日復習）

#### 步驟五：建立持續複習的習慣

**最佳複習時機**：
- **早晨**：大腦清醒，適合學習新內容
- **睡前**：有助於記憶鞏固
- **碎片時間**：通勤、等待時的5-10分鐘

**複習策略**：
- 先完成當日的復習卡片（維持舊知識）
- 再學習新卡片（擴展新知識）
- 遇到困難卡片時，回到原始情境重新建立記憶錨點

### 進階技巧：AI 協助製作 Anki 卡片

你可以請 AI 幫你自動生成符合科學原理的 Anki 卡片：

#### 👉 實際可用的 Prompt 範本：

```
請幫我製作 Anki 學習卡片，單字是 "______"

卡片格式：
正面：
- 情境引導問題
- 視覺提示（描述圖片內容）
- 清楚的提問

背面：
- 單字 + 音標
- 記憶錨點（圖像+情境+故事）
- 實用例句
- 語意網路連結（3-5個相關單字）

請確保內容適合間隔複習使用。
```

**範例輸出**：AI 會根據你的要求生成完整的卡片內容，你只需要複製貼上到 Anki 中即可。

### Anki 學習週期範例

以「dumbbell」為例，看看 Anki 如何安排復習：

**第1天**：初次學習 → 答對 → 隔1天復習
**第3天**：復習 → 答對 → 隔4天復習  
**第7天**：復習 → 答對 → 隔10天復習
**第17天**：復習 → 答對 → 隔25天復習
**第42天**：復習 → 答對 → 間隔持續拉長...

如果某次答錯，間隔會重新縮短，確保在真正掌握前不會被遺忘。  

透過 Anki 的間隔複習系統，你不需要手動計算複習時間，也不用擔心會忘記重要的單字。系統會在最佳時機提醒你複習，讓學習變得更有效率、更持久。從今天開始，讓科學為你的英文學習護航吧！

## 用科學理論學語言：從零散知識到系統學習
